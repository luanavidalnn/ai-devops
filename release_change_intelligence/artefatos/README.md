# Artefatos do programa Release Change Intelligence

Tudo o que for **desenvolvido e aprovado** para operacionalizar o Ponto 1 deve ficar aqui (ou ser referenciado daqui), para nao espalhar ficheiros soltos pelo repositorio.

## Estrutura prevista

```
artefatos/
  README.md          (este ficheiro)
  pt/
    01_CHECKLIST_COLETA_RELEASE.md      (Fase 1: configuracao + coleta + pos)
    02_PROMPT_ANALISE_RELEASE_V1.md     (Fase 2: prompt analise V1)
    03_FICHA_POS_RELEASE.md                 (Fase 3: retro pos-janela)
  en/
    01_CHECKLIST_RELEASE_COLLECTION_AND_SETUP.md
    02_PROMPT_RELEASE_ANALYSIS_V1.md
    03_POST_RELEASE_PILOT_NOTE.md
```

## Regras

- Versoes de prompts: sufixo `_V1`, `_V2` no nome ou tabela de versao no topo do ficheiro.
- Nunca commitar dados reais de cliente, credenciais ou exports com PII; usar exemplos ficticios ou hashes/IDs.
