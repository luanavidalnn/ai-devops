# Prompt Template Library (Approved)

## Purpose
Provide standardized prompt templates for repeatable operational analysis.

## Templates
- Incident triage
- Hypothesis generation
- Handoff summary
- Postmortem draft
- Release risk briefing

## Standard Prompt Sections
- Context
- Evidence provided
- Required output format
- Confidence statement
- Data gaps and next checks

## Approval Workflow
1. Draft by platform team.
2. Security and operations review.
3. Publish as approved template.
4. Revalidate biweekly.
