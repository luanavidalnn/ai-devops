# Biblioteca de Templates de Prompt (Aprovados)

## Objetivo
Disponibilizar templates de prompt padronizados para analises operacionais repetiveis.

## Templates
- Triagem de incidente
- Geracao de hipoteses
- Resumo de handoff
- Rascunho de postmortem
- Briefing de risco de release

## Secoes Padrao do Prompt
- Contexto
- Evidencias fornecidas
- Formato de resposta exigido
- Declaracao de confianca
- Lacunas de dados e proximas checagens

## Fluxo de Aprovacao
1. Rascunho pelo time de plataforma.
2. Revisao de seguranca e operacoes.
3. Publicacao como template aprovado.
4. Revalidacao quinzenal.
